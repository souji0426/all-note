# 環境構築メモ
PC買い替えなどで環境が変わったときのためのメモ。<br>
TeXのインストール方法：[ここにお世話になっている](https://did2memo.net/2016/04/24/easy-latex-install-windows-10-2016-04/)<br>
エディタ：Atom <br>
TeXLiveなど色々と試したけど、これがやっぱり好き。

インストールするパッケージ一覧や設定など
- 個人設定のautocompleteCharactersに「"$$"」を加える<br>
これで「$」が自動補完されて便利。

- コア設定のファイルエンコーディングを「shift-jis」に<br>
日本語ファイル名をWindowsで扱うならこの方が楽かも。

- パッケージ：Auto Encoding For Ruby<br>
ただ周辺ツールなどはRubyで作るため、このパッケージを入れてRubyファイルは「utf8」で開くようにしておく。

- パッケージ：Activate Power Mode<br>
退屈なコーディングに癒しを（時々煩わしい）。

- パッケージ：Browser Refresh On Save<br>
Atomを起動したとき、前回開いていたタブなどを復元してくれる。

- パッケージ：Document Outline<br>
右側にアウトラインを表示してくれる。多くのセクションを含むTeXファイルを編集するときとか便利

- パッケージ：Japanese Menu<br>
使用言語を日本語にしてくれる。英弱の私向け。

- パッケージ：Ruby Block<br>
Rubyコーディング用に。

Perl <br>
作業効率化ツールのためのプログラミング言語。
バージョンはv5.24.3。ファイル操作や簡単なファイルコピーなどは基本的にこれを使用。<br>
またPerlを触るときは上の設定ファイルの「$」の自動補完を切っておいたほうがいい。
Active Perlにしておき、PPMを使って、以下のライブラリをインストールしておくこと。

- Config::Tiny<br>
ツールの設定ファイルはini形式なので、それをパースしてくれるこのライブラリを使っている。
